# russell logs
查看任务的运行日志。

## 查看帮助
```bash
russell logs --help
```
输出：
```
Usage: russell logs [OPTIONS] ID

  Print the logs of the run.

Options:
  -t, --tail  Stream the logs
  --help      Show this message and exit.
```

## 命令
```bash
russell logs [OPTIONS] ID
```

## 选项

|选项|默认值|描述|
|---|---|---|
|ID||任务ID|
|--url, -u||只打印URL|。；
