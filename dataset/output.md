# 查看数据集内容


* 从数据集版本界面点击查看文件

```
   URL:  http://russellcloud.com/<user>/dataset/<dataset_name>/versions
```

![](/asserts/img/output_dataset_1.png)

* 使用命令行按版本ID查看内容（自动跳转默认浏览器显示数据集）

```
    russell data output <version_id>
```
