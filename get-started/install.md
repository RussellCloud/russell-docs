## 基础安装

*russell-cli* 是基于Python的命令行工具，我们在终端通过它来进行启动、监控任务或者上传数据集等操作。

- 安装

```bash
pip install -U russell-cli
```

- 高级用法

```bash
russell --help
```

详情查看[命令行工具](/cli.md)

### 系统兼容
- Windows
- MacOs
- Linux

### Python兼容
- Python2.7
- Python3.5

### 小技巧
