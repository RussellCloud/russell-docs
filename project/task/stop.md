## 停止任务

在很多情况下，你可能需要手动停止任务，比如在jupyter模式下或者仅仅是刚才跑的任务已经没有意义了，这时候你需要在终端执行下面的命令：

```
$ russell stop <task_id>
```

task\_id 即运行时返回的 RUN\_ID,你也可以在网站上个人首页的任务列表中找到对应的任务id。
